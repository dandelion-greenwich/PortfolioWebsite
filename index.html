Projects

Coco Dash

3D game inspired by PAC-MAN using Unity engine, created as a group project during January 2024 - March 2024. I worked on the project as lead programmer, where I was responsible for advanced programming features and solving git conflicts.

What I learned and implemented:

  • Player and camera movement
  • Enemy AI, NavMesh
  • Character abilities
  • Game loop
  • State machine using enumerators
  • Git version control

The project was showcased at Digital Shark Expo at University of Greenwich (2024) and received a 1st class grade.
